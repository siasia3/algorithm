# [Silver II] 쇠막대기 - 10799

[문제 링크](https://www.acmicpc.net/problem/10799)

## 📌 문제 설명

괄호 문자열로 표현된 쇠막대기와 레이저 배치가 주어진다.

- `()` → 레이저
- `(`, `)` → 쇠막대기의 시작과 끝

레이저는 현재 열려 있는 모든 쇠막대기를 자르고,  
막대기의 끝은 하나의 조각을 만든다.

잘려진 쇠막대기 조각의 총 개수를 구하자.

---

## 💡 풀이 아이디어

스택을 이용해 **현재 열려 있는 쇠막대기의 개수**를 관리한다.

### 핵심 로직

- `(` → 스택 push
- `)` →
    - 직전이 `(` → **레이저**
        - 스택 pop
        - `stack.size()` 만큼 조각 추가
    - 직전이 `)` → **막대기 끝**
        - 스택 pop
        - `+1`

---

## 🔥 핵심 포인트

- 레이저일 때: `현재 열려 있는 막대기 수 만큼 증가`
- 막대기 끝일 때: `+1`

---

## 🧠 시간 복잡도

- O(N)
